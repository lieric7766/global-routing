使用之程式語言：< C++ >     
使用之編譯平台：< Linux GNU g++ >    
各檔案說明：    
```
.
|    main.cpp
|    global_routing.cpp
|    global_routing.h
|    README.md
```

編譯方式說明：     
> _主程式：_  
> _在 global-routing/ 這個資料夾下指令 :_ `g++ global_routing.cpp main.cpp -o main`  
> _即可產生 main 執行檔_     

執行、使用方式說明：    
> _主程式：_  
> _compile 完成後，在 /global-routing/ 目錄下會產生一個 main 的執行檔_  
> _執行檔的命令格式為 :_  `./main [input file name] [output file name]` 
> _ex:_ `./main 12.in 12.out`
